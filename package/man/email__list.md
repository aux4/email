#### Description

The `list` command group provides subcommands for listing emails in your mailbox. You can list all emails or filter by sender address. Results are displayed in a table format showing UID, date, sender, and subject.

#### Usage

```bash
aux4 email list <subcommand>
```

Available subcommands: all, from

#### Example

```bash
aux4 email list all
aux4 email list from user@example.com
```
